Interactive Map Explorer
-A responsive web application that displays points of interest on an interactive map using Leaflet.js.

Features
-Interactive Map: View locations on a customizable map
-Smart Markers: Color-coded pins for different categories (landmarks, museums, parks, restaurants)
-Search & Filter: Find locations by name or filter by category
-Location Details: Click markers to see descriptions and images
-Your Location: Find yourself on the map with geolocation
-Mobile Friendly: Works perfectly on all devices
